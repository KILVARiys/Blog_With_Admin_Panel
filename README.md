# Blog_With_Admin_Panel

# 📝 Блог с админ-панелью

Простой, но функциональный движок блога на Flask с защищенной админ-панелью. Позволяет создавать, редактировать и удалять посты через удобный интерфейс.

## ✨ Возможности

### Публичная часть
- 📄 Список всех постов на главной странице
- 🔍 Просмотр отдельного поста с полным содержанием
- 📱 Адаптивный дизайн
- 🕒 Сортировка постов по дате публикации (новые сверху)

### Админ-панель
- 🔐 Защита паролем с использованием Flask-Login
- ✏️ CRUD операции с постами (создание, чтение, обновление, удаление)
- 🛡️ CSRF защита всех форм
- 💬 Flash-сообщения с уведомлениями
- 🗑️ Подтверждение удаления поста

### Технические особенности
- 🗄️ SQLite база данных (легкая, не требует отдельного сервера)
- 🔒 Безопасное хеширование паролей (werkzeug.security)
- 🌐 Переменные окружения для конфиденциальных данных
- 📦 Простая структура проекта

## 🚀 Технологии

- **Backend:** Python 3.8+, Flask
- **База данных:** SQLite3
- **Аутентификация:** Flask-Login
- **Безопасность:** Flask-WTF (CSRF), werkzeug.security
- **Frontend:** HTML5, CSS3, Jinja2 templates

## 📋 Требования

- Python 3.8 или выше
- pip (менеджер пакетов Python)

## 🛠️ Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/your-username/blog-with-admin-panel.git
cd blog-with-admin-panel
```

### 2. Создание виртуального окружения

```bash
# Windows
python -m venv venv
venv\Scripts\activate

# Linux/Mac
python3 -m venv venv
source venv/bin/activate
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Настройка переменных окружения

Создайте файл `.env` в корне проекта:

```env
# Секретный ключ Flask (обязательно измените!)
SECRET_KEY=ваш-секретный-ключ-здесь

# Данные администратора (создаются автоматически при первом запуске)
ADMIN_USERNAME=admin
ADMIN_PASSWORD=ваш-сложный-пароль

# Настройки базы данных
DATABASE_PATH=blog.db

# Режим отладки (True для разработки, False для продакшена)
FLASK_DEBUG=True
```

### 5. Запуск приложения

```bash
python app.py
```

Приложение будет доступно по адресу: `http://127.0.0.1:5000`

### 6. Вход в админ-панель

1. Перейдите на страницу входа: `http://127.0.0.1:5000/login`
2. Введите данные администратора из файла `.env`
3. После входа вы попадете в админ-панель

## 🔧 API Эндпоинты

### Публичные
| Метод | URL | Описание |
|-------|-----|----------|
| GET | `/` | Главная страница (список постов) |
| GET | `/post/<int:post_id>` | Просмотр отдельного поста |

### Аутентификация
| Метод | URL | Описание |
|-------|-----|----------|
| GET/POST | `/login` | Вход в админ-панель |
| GET | `/logout` | Выход из системы |

### Админ-панель (требуется авторизация)
| Метод | URL | Описание |
|-------|-----|----------|
| GET | `/admin` | Панель управления |
| GET/POST | `/admin/post/new` | Создание нового поста |
| GET/POST | `/admin/post/<int:id>/edit` | Редактирование поста |
| POST | `/admin/post/<int:id>/delete` | Удаление поста |

## 🔒 Безопасность

- ✅ Пароли хранятся в хешированном виде (не в открытом доступе)
- ✅ CSRF защита всех форм
- ✅ Секретные ключи в переменных окружения
- ✅ Защита админ-панели через Flask-Login
- ✅ SQL-инъекции предотвращены через параметризованные запросы

## 🚀 Деплой

### На сервере (Ubuntu/Debian)

1. Установите Python и необходимые пакеты:
```bash
sudo apt update
sudo apt install python3 python3-pip nginx
```

2. Настройте WSGI сервер (gunicorn):
```bash
pip install gunicorn
gunicorn -w 4 app:app
```

3. Настройте Nginx как reverse proxy

### На платформах

- **Heroku:** Добавьте `Procfile` и настройте переменные окружения
- **PythonAnywhere:** Простая загрузка через git
- **DigitalOcean:** Развертывание на VPS

## 🐛 Известные проблемы

- При первом запуске необходимо вручную удалить роут `/setup-admin` после создания админа
- Для продакшена рекомендуется изменить `FLASK_DEBUG=False`
